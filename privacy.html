Политика обработки персональных данных

Настоящая политика описывает порядок обработки персональных данных, которые посетитель сайта FIRST LINE передаёт через форму обратной связи.

1. Какие данные обрабатываются

Через форму на сайте обрабатывается номер телефона, указанный пользователем для обратной связи по вопросам аренды коммерческих помещений.

2. Цель обработки

Данные используются только для связи с пользователем, уточнения запроса, подбора помещения и организации показа.

3. Правовое основание

Отправляя форму, пользователь выражает согласие на обработку указанного номера телефона для обработки заявки.

4. Передача третьим лицам

Данные не передаются третьим лицам, за исключением случаев, когда такая передача требуется по закону.

5. Срок хранения

Данные хранятся в течение срока, необходимого для обработки заявки и последующей коммуникации по запросу пользователя.

6. Защита данных

Принимаются разумные организационные и технические меры для защиты данных от несанкционированного доступа.

7. Отзыв согласия

Пользователь может запросить прекращение обработки персональных данных, обратившись к владельцу сайта по контактам, указанным на сайте или в переписке.
