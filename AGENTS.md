# FIRST LINE site

## Project type
Static HTML/CSS/JS landing with PHP feedback form.

## Rules
- Do not add React, Next.js, Vite, npm, package.json, bundlers or frameworks.
- Do not use external CDN or Google Fonts.
- Keep the approved visual design from `index.html`.
- Feedback form must use `send.php` with `method="post"`.
- Keep anti-spam honeypot and server-side phone validation in `send.php`.
- Do not expose detailed technical mail errors to users.
- Images are referenced from `assets/images/object-1.jpg` through `assets/images/object-5.jpg`, but do not commit image files or other binaries; keep the directory with `assets/images/.gitkeep`.
- Target hosting: Beget PHP hosting, upload project files to `public_html`.
